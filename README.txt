Pour utiliser le programme, il faut lancer interface.py
Il faut utiliser Python 2.*, et avoir installé OpenCV pour python et PySide
Attention : il faut se placer dans le répertoire du code avant de le lancer, sinon il ne trouvera pas modules.js

OpenCV peut mal fonctionner sous Windows, le code a été testé sans problème sous linux.

Seul le module Suma peut être chargé, les autres ne sont là que pour remplir la liste (modifiable en éditant modules.js, la syntaxe est assez simple)

Le répertoire faces doit également être situé au même endroit que le code exécuté.
